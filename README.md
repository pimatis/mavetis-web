# Mavetis Web

Official website for [Mavetis](https://github.com/pimatis/mavetis). Enterprise-grade, local-first Git security scanner.

## Installation

```bash
bun install
```

## Development

```bash
bun run dev
```

## Build

```bash
bun run build
```

## Preview

```bash
bun run preview
```

## Type Check

```bash
bun run check
```
